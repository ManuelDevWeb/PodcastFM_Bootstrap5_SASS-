游릭 Configuracion Inicial del Proyecto

1. Inicializar proyecto: npm init

2. Instalar sass: npm i -D sass gulp-sass

3. Instalar Bootstrap: npm i -D bootstrap

4. Crear la carpeta: src/scss/ui

5. Crear el archivo: src/scss/ui/_index.scss

游릭 Gulp

6. Instalar gulp (Compilar hojas de estilos y guardarla en otro lugar): npm i -D gulp

7. Instalar gulp globalmente: npm i -g gulp-cli

8. Crear el archivo: gulpfile.js

9. Instalar la dependencia: npm i -D gulp-imagemin@7.1.0

10. Compilar tarea creada en el archivo gulpfile: gulp dev (dev es el nombre de la tarea)

11. Compilar tareas por defecto: gulp
